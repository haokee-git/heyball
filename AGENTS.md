使用 raylib C++ 制作一个 2D 视角的 3D 游戏，你可以使用 D:\msys64。注意 PATH 中的 g++ 不是 MYSY2 中的。

你需要制作一个台球游戏，支持两名玩家轮流操作对战。规则为：中式八球。你需要上网搜寻资料已了解详细规则。

镜头正俯视球台。你需要制作不同的 3D 球在 2D 下的投影图像。半色球、全色球、球上的数字等等全部需要仔细做出来。帧数尽量高。

击杆后，球与球、球与库之间的碰撞、摩擦等你需要用真实的物理引擎详细模拟。支持在右上角选择高低杆、左右塞。默认中杆不加塞。

操作方式：用户拖动鼠标来控制杆击白球的角度。用户鼠标中键（滚轮）下滑即加力，杆随即向后动；向前即击球。

背景纯黑色，UI 沉稳、高雅且收敛。用等线非衬线字体。
